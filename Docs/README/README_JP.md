# SVSラボアライナー

**歌声合成ラボアライナー**

歌声合成（SVS）ワークフローのための完全スタンドアロンWebアプリケーションです。入力音声とテキストを、歌声合成ソフトウェアで使用可能なプロジェクトファイルに変換します。

このアプリケーションの目的は、TTS（テキスト読み上げ）機能を作成したり、音声合成エンジン自体を置き換えたりすることではなく、歌手が「上手に話しているように見せる」ための架け橋となることです。

- まず、**[MFA](https://github.com/MontrealCorpusTools/Montreal-Forced-Aligner)**を使用して音声とテキストを自動的にアライメントし、`.lab`ファイルを生成します。

- 次に、**[PyWORLD](https://github.com/JeremyCCHsu/Python-Wrapper-for-World-Vocoder)**を使用して基本周波数（F0）を抽出します。


最後に、音声合成プロジェクトファイルを出力し、ソフトウェア内で編集や再生を行います。

## ✨ 主な機能

* **アノテーションのみ（高速）**
MFA自動アライメントのみを実行し、`.lab`ファイルを高速に生成します。

* **完全処理（アノテーション + F0 + プロジェクトファイル）**
アノテーション、ピッチ抽出を自動的に完了し、音声合成プロジェクトファイルを出力します。

* **プロジェクト生成のみ（WAV + LAB）**
既存の`WAV + LAB`ファイルに基づいてプロジェクトファイルを直接生成するため、再アノテーションは不要です。

* **複数のピッチ抽出アルゴリズムに対応**
**PyWORLD、CREPE、RMVPE**の3つのF0抽出アルゴリズムを内蔵しており、特定のニーズに応じて異なるアルゴリズムを選択できます。

* **連続ピッチ書き込みに対応**
純音のみを生成するか、連続F0カーブをプロジェクトに書き込むかを選択でき、よりリアルなピッチ変化を再現できます。

* **長尺音声処理に対応**
楽曲全体、長尺のスピーチなど、様々なシーンに対応する長尺音声ファイルの処理が可能です。

* **複数のテキスト変換方法に対応**
「プロジェクトのみ生成（WAV + LAB）」モードでは、以下のオプションを選択できます。

* 変換なし

* 子音結合

* ひらがな

* カタカナ

* **複数の音声フォーマットに対応**
`WAV / MP3 / FLAC / M4A / AAC`

* **多言語テキストアライメントに対応**

* **LABファイルのエクスポートに対応**

* **複数のアライメントアルゴリズムに対応**
MFA、WhisperX、Qwen3-ASR、Qwen3-FAの4つのアライメント手法を内蔵しており、ユーザーはそれぞれのニーズに合わせて最適なアルゴリズムを選択できます。

* **シンプルで使いやすいWebインターフェースを提供**

## 📌 適用シナリオ

- 音声合成またはTTSオーディオを既にお持ちで、注釈を素早く生成したい場合。

- 音声プロジェクトを音声合成ソフトウェアにインポートして、さらに編集したい場合。

- `.lab`、F0、およびプロジェクトファイルを一括生成したい場合。

- 「話す」と「歌う」の間に編集可能な中間レイヤーを作成したい場合。

---

## 🛠️ 使用方法

1. オーディオファイルとテキストを入力します。


2. 自動時間アライメントには、**[MFA](https://github.com/MontrealCorpusTools/Montreal-Forced-Aligner)**を使用してください。

3. `.lab` ファイルを生成します。アノテーションファイル

4. **[PyWORLD](https://github.com/JeremyCCHsu/Python-Wrapper-for-World-Vocoder)** を使用してF0ピッチを抽出します。

5. 処理モードに応じてプロジェクトファイルを出力します。

6. 結果を音声合成ソフトウェアにインポートして、さらに使用します。

## 🔧 処理モード

### 1. アノテーションのみ (クイック)

MFAアノテーションのみを自動的に実行し、.labファイルを生成します。


### 2. 完全処理（アノテーション + F0 + プロジェクトファイル）

MFAアライメント、F0抽出を行い、プロジェクトファイルを生成します。

### 3. プロジェクトのみ生成（WAV + LAB）

既存のWAV + LABファイルを使用してプロジェクトファイルを生成します。完全なアノテーション処理は繰り返されません。

## 📖 使用手順

### アノテーションのみ（クイック）

1. アプリケーションを開く

2. 音声ファイルをアップロードする

3. テキストを入力する

4. 言語を選択する

5. 「アノテーションを開始」をクリックする

6. 生成された.labファイルをダウンロードする

### 完全処理（アノテーション + F0 + プロジェクトファイル）

1. 音声ファイルをアップロードする

2. 1. テキストを入力

2. 言語を選択

3. 処理モードを「フル処理」に選択

4. 「処理開始」をクリック

5. 生成されたプロジェクトファイルをダウンロード

### プロジェクトのみ生成（WAV + LAB）

1. WAVファイルと対応するLABファイルを準備

2. 「プロジェクトのみ生成」を選択

3. WAVファイルとLABファイルをアップロード

4. 「生成」をクリック

5. プロジェクトファイルをダウンロード

### 📝 出力内容
処理モードに応じて、アプリケーションは以下のファイルを出力します。

- .lab タイムアラインメント済みファイル

- F0ピッチ情報

- 音声合成プロジェクトファイル

### 📌 機能制限

本ツールは「SVSワークフロー補助ツール」としてのみ使用されることを想定しており、以下の機能は備えていません。

- 音声合成（TTS / SVS）は実行しません。

- 音声バンクモデルの学習または変更は行いません。

- 音声合成ソフトウェアの代替となるものではありません。
