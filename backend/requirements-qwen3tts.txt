# requirements-qwen3tts.txt
# qwen3tts_server.py 独立服务依赖（不要装进主 .mfa_env，也不要和
# .qwen3_env / .nemo_env 共用！）
#
# 用法：
#   conda create --prefix ../.qwen3tts_env python=3.12 -y
#   conda activate ../.qwen3tts_env
#   pip install -r requirements-qwen3tts.txt
#   python qwen3tts_server.py
#
# 说明：
#   这是 Qwen3-TTS（CustomVoice / VoiceDesign / Base-VoiceClone 三套
#   12Hz checkpoint）跑在独立进程里所需的全部依赖。主 Flask 应用
#   （app.py 所在的 .mfa_env）只通过 HTTP 调用 127.0.0.1:5003，完全不
#   需要装 qwen-tts / 对应版本的 transformers 这一套。
#
#   之所以单独建环境：qwen-tts 官方要求较新的 transformers 版本（用于
#   支持 Qwen3-TTS-Tokenizer-12Hz 的模型结构），与 qwen3_server.py
#   （qwen-asr 所在环境）锁定的 transformers>=4.40.0 不一定是同一个精确
#   版本，装进同一个 venv 容易互相"打架"，把对方需要的版本降级。
#
#   官方推荐 Python 3.12（见 https://github.com/QwenLM/Qwen3-TTS
#   Environment Setup 一节），这里与之保持一致。
